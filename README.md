# membivo – Releases

Öffentliches Release-Repository für **membivo** (Check-in-System). Enthält ausschließlich die vom CI automatisch gebauten Release-Artefakte – der Quellcode liegt im privaten Repo `dennisentruencer/membivo`.

- `membivo-<version>.tar.gz` – portables Update-Bundle für den In-App-Updater
- `Membivo-<version>.pkg` – macOS-Erstinstallation

Releases pusht der Workflow `release.yml` im Source-Repo bei jedem `v*`-Tag hierher.
